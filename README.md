# C# Course — from "what is a program?" to job-ready

A **free, beginner-friendly C# course**. It starts right at the beginning —
*"what is a program?"* — assuming you've never written code, and builds you up
step by step to writing real .NET applications and being ready for a developer
job. It's a great fit for complete beginners and for people coming from Python.

Every topic gives you **lessons to read**, a **quiz** to check you understood,
**hands-on practice problems**, and **projects** to build. Your progress is saved
as you go, so you can stop any time and pick up exactly where you left off.

**👉 Course: https://javvajikiran167.github.io/csharp-course/**

## Course contents

**1. Programming Foundations** — *8 lessons · 11 practice problems*
What is a program? · What is a programming language? · How computers store data ·
Variables & types · How programs decide and repeat · Algorithms · From source
code to a running program · Your toolkit (installing .NET, first program)

**2. Foundations of C#** — *9 lessons · 21 practice problems*
Why C# · How C# runs · Setting up · Variables & types · Operators · Strings ·
Console input & output · Constants & comments · Mini-project: tip calculator

**3. Control Flow** — *7 lessons · 21 practice problems*
Booleans & comparisons · if / else / else if · Ternary & null-coalescing ·
switch statement · switch expressions & pattern matching · Short-circuit
evaluation · Mini-project: number guessing game

**4. Loops & Iteration** — *7 lessons · 21 practice problems*
while loops · for loops · foreach loops · do-while · break, continue & return ·
Nested loops & complexity · Mini-project: FizzBuzz & multiplication table

**5. Methods & Reusability** — *6 lessons · 10 practice problems*
Defining & calling methods · Pass by value vs by reference · Method overloading ·
Default & named arguments · params arrays · Local & expression-bodied methods

**6. Collections** — *7 lessons · 10 practice problems*
Arrays · Multidimensional & jagged arrays · `List<T>` · `Dictionary<TKey,TValue>` ·
`HashSet<T>` · `Stack<T>` & `Queue<T>` · Iterating, sorting & filtering

**7. Object-Oriented Programming** — *11 lessons · 12 practice problems*
Classes & objects · Constructors · Properties (get/set/init) · Access modifiers &
encapsulation · Static members · Inheritance · virtual / override / sealed ·
Abstract classes · Interfaces · Records & value equality · Structs & enums

**8. Exception Handling** — *5 lessons · 10 practice problems*
try / catch · finally & using · Throwing & re-throwing · Custom exception types ·
Exception best practices

**9. Files & Serialization** — *5 lessons · 11 practice problems*
Reading & writing text files · StreamReader & StreamWriter · Paths & the file
system · JSON with System.Text.Json · Working with CSV

**10. C# Conventions & Style** — *1 lesson · 3 practice problems*
Why C# has conventions

**11. Delegates, Events & Lambdas** — *5 lessons · 10 practice problems*
Delegates (methods as values) · Func, Action & Predicate · Lambdas & closures ·
Events & the event keyword · Callbacks & functional patterns

**12. Generics & LINQ** — *8 lessons · 10 practice problems*
Generic methods · Generic classes · Type constraints · LINQ method syntax · LINQ
query syntax · GroupBy, Join & aggregation · Deferred execution · Mini-project:
sales data analyzer

**13. Async & Await** — *6 lessons · 10 practice problems*
Why async (blocking vs non-blocking) · Task & `Task<T>` · async / await syntax ·
Common async pitfalls · CancellationToken · Mini-project: parallel URL fetcher

**14. Databases & EF Core** — *7 lessons · 15 practice problems*
What is an ORM? · DbContext & DbSet · Migrations · Querying with LINQ to EF ·
Insert, update, delete · One-to-many & many-to-many · Mini-project: blog with SQLite

**15. Building a Web API** — *8 lessons · 27 practice problems*
Minimal API hello world · Routing & HTTP verbs · Controllers vs Minimal API ·
Model binding & validation · Dependency injection · Middleware pipeline ·
Authentication & authorization · Mini-project: tasks REST API

**16. Testing & Design Patterns** — *6 lessons · 18 practice problems*
xUnit basics · Arrange-Act-Assert · Mocking with Moq · Test-driven development ·
SOLID principles · Singleton, Factory, Strategy, Observer

**17. Job Prep & Capstone** — *6 lessons · 18 practice problems*
Common algorithm problems · A mock interview walkthrough · Reading & reviewing
other people's code · Capstone spec (tasks app) · Build & iterate · Where to go next
